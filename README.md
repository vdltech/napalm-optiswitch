# napalm-optiswitch

Napalm Driver for MRV/Adva Optiswitch devices

## Supported functions

* get_facts
* get_interfaces
* get_interfaces_ip
* get_lldp_neighbors
* get_lldp_neighbors_detail
* get_vlans
* get_interfaces_vlans
* get_mac_address_table
* get_config
* load_merge_candidate
* discard_config
* commit_config

## Supported Models

* OptiSwitch 904
* OptiSwitch 906
* OptiSwitch 940
* OptiSwitch V8
